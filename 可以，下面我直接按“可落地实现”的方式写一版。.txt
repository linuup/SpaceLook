**五、QmlPreviewHost 设计**

**十一、PreviewHost 具体改法**

建议 `PreviewHost::showPreview(info)` 变成：

1. 根据 `performanceMode` 选后端
2. 如果当前后端变了，隐藏另一套宿主
3. 把 `info` 传给当前宿主
4. 当前宿主负责真正显示

伪代码：

```cpp
void PreviewHost::showPreview(const HoveredItemInfo& info)
{
    const bool useQml = SpaceLookUiSettings::instance().performanceMode();
    applyBackendVisibility(useQml);

    if (useQml) {
        m_qmlHost->showPreview(info);
    } else {
        m_widgetHost->showPreview(info);
    }
}
```

`stopPreview()` 同理。
